# Docker Usage

The Docker tutorial explains how to build a RoboKudo container image from the RoboKudo repository Dockerfile, allow local X11 display access, run the container with host networking and NVIDIA GPU support, and start RoboKudo inside it.

The page notes that the tutorial is only tested for ROS 1. It also notes that `roscore` no longer exists in ROS 2, even though a ROS 2 run command is shown.
